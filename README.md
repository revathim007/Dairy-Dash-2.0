# Dairy Dash 🥛

A full-stack web application for managing milk delivery subscriptions for customers and administrators.

This system allows customers to subscribe to daily milk delivery while administrators manage products, customers, subscriptions, and billing.

---

# 🚀 Features

## Admin Dashboard

* Manage milk products
* Manage customers
* Manage subscriptions
* View and track orders
* Billing management

## Customer Dashboard

* Browse milk products
* Add products to cart
* Subscribe for daily milk delivery
* View order history
* Manage profile
* View billing details

# ⚙️ Installation

## Backend Setup

```
cd backend
pip install -r requirements.txt
python manage.py runserver
```

Backend runs on:

```
http://127.0.0.1:8000
```

---

## Frontend Setup

```
cd frontend
npm install
npm start
```

Frontend runs on:

```
http://localhost:3000
```
